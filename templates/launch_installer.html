{% if os_type == 'mac' %}

Installing on macOS

1
Open the downloaded zip file

Locate the DearTeddyInstaller-Mac.zip file in your Downloads folder and double-click to extract it.

2
Drag to Applications

Drag the "DearTeddy.app" file to your Applications folder.

3
Handle Security Warning

When first opening the app, if you see a security warning:

  1. Do NOT click "Move to Bin" - this will delete the app
  2. Instead, close the warning dialog by clicking "Cancel"
  3. Go to System Preferences > Security & Privacy
  4. Look for the message about DearTeddy being blocked
  5. Click "Open Anyway" to authorize the app
  6. When prompted again, click "Open"

Alternative Method: Right-click (or Control-click) on the app icon, select "Open", then click "Open" in the dialog. This often bypasses the stricter security checks.
